Welcome to Doodle Launcher

Thank you for downloading Doodle Launcher. Let's start!

Shortcut to your favorite pages

Like a bookmark list, create shortcuts linking to your frequently visited websites and access them with a single click.

Organize your shortcuts into categories

Like file cabinets, you can categorize your shortcuts into toggleable tabs that you can organize into.

Easily edit shortcuts

With a click of a button, you can edit or delete shortcuts and categories through the Shortcut Editor.

Customize your home screen

Give your shortcut library character that is unique to you. Doodle Launcher lets you customize a lot of aspects on your home page, such as the colors and font.

Search quick

Using the launcher and have to search for something? Instead of opening a new tab and searching from there, pull up the Quick Search Bar using Ctrl+Shift and search from there, and the launcher will automatically open a tab with your search query using the search engine of your choice.

What's your name?

Start your browsing through the launcher with a personalized greeting. Do you want to have your name included in the home page greeting? If yes, please write your name in the input box below. This can be changed in the settings.

Before you start...

It's important to know how the launcher works and how it handles information.

How your data is stored

Doodle Launcher stores all saved data into your browser's internal storage. Data written and saved while your using the launcher will not carry over when you switch browsers.

What system the launcher use

The launcher uses JavaScript to do most of the functions and features.

Vulnerabilities

Because all data is stored into the browser's internal storage, it is easy for other websites to execute a similar JavaScript code to access these keys. This is a vulnerability that is hard to solve using the current system without using some kind of online storage alternative / cloud storage.

Do not include any sensitive information

It is a bad idea to include any sensitive information such as passwords, addresses, etc. into any input boxes within the launcher. Data stored can easily be accessed by websites you visit and may use it for their own heinous intentions.

Thank you!

That's all for now. Click Start now to begin using Doodle Launcher.

Public Preview 4 Update

Thank you for updating.

Here's what changed:

Additions

  • Added a way to re-order categories and shortcuts in Shortcut Editor. This is still experimental and may break unexpectedly. Please report any bug you may encounter.
  • Added the preset "Garden of Words" and "Underwater".
  • Added wallpaper previews and a better preset selector tab on Settings > Appearance.
  • Added a "Go to Bottom" button that appears when the "Back to Top" button is hovered.
  • Added an "Open Category Navigation" button that appears when the "Back to Top" button is hovered.
  • Added a behavior setting for Shortcut Editor that lets you choose how windows are closed and how input fields are cleared.

Changes

  • Reworked the shortcut generation system and the home page should start a little bit faster now.
  • Shortcut Editor Add Item subwindow will automatically close after creating an item by default.
  • Shortcut Editor input fields will automatically clear content after creating an item by default.
  • The 6 checkboxes at the top of the Behavior tab in Settings > Behaviors had been moved to its own "Display" category.
  • Presets now includes the wallpaper.
  • Launcher will now check for internet connection every second instead of at every session.
  • Removed the version and copyright text from the Session Screen.
  • Removed the backdrop filter on the loading screen and will now display a solid color instead of a blurred background.
  • The Category Navigation Sidebar now has a close button.
  • Changed the Information tab icon in Settings > Information
  • Fixed toggleable elements not showing up on first click.
  • Fixed a bug where the opening animation doesn't play or bugs out.
  • Fixed a bug where more than one item is deleted at a time in some occasions on Shortcut Editor.
  • Fixed a bug where the default wallpaper doesn't show up when loading the default setting values.
  • Fixed a bug where default behaviors doesn't set properly when loading the launcher for the first time.
